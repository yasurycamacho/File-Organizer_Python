# Organizador de archivos - Python Automation

Este proyecto es una herramienta de automatización desarrollada en **Python** diseñada para optimizar la gestión de archivos. El script clasifica y organiza automáticamente documentos basándose en sus extensiones, moviéndolos a carpetas categorizadas.

## Funcionalidades
* **Clasificación Automática:** Identifica extensiones (PDF, JPG, DOCX, etc.) y las agrupa.
* **Gestión de Directorios:** Crea carpetas automáticamente si no existen.
* **Optimización de Tiempo:** Reduce el desorden digital en segundos.

## Tecnologías Usadas
* **Lenguaje:** Python 3.x
* **Librerías:** `os` y `shutil` (Manejo de sistema de archivos).

## Cómo usarlo
1. Descarga el archivo `carpeta_inteligente.py`.
2. Ejecuta el script en la carpeta que desees organizar.
